Detailed Description

Class holding true track information.

By default,

  • No Cherenkov photons are saved in the output file (in fact no photons are stored at all. The exception is what Geant terms as gammas - basically high energy photons)
  • Everything that creates a photon that creates a true PMT hit is saved, along with all its ancestors
